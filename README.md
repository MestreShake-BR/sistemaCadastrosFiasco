# sistemaCadastrosFiasco

## Descrição
O **sistemaCadastrosFiasco** é um sistema de cadastros desenvolvido em **Java Spring**, criado inicialmente com base em uma aula do YouTuber Fiasco. Ele funciona como um sistema CRUD (Create, Read, Update, Delete) para gerenciamento de dados de empresas e seus recursos.  

O projeto tem caráter **didático**, sendo usado como base para prática de desenvolvimento com **Java Spring**, mas também é pensado como um **sistema de longo prazo**, capaz de evoluir para um gestor de empresas completo.

## Tecnologias Utilizadas
- Java 21
- Spring Boot
- Spring Data JPA
- Banco de dados H2
- Maven
- REST API
